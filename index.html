APT Detector
Implemented a research paper dealing with MORSE, a framework capable of detecting Advanced Persistent Threats in a real-time fashion while yielding compact scenario graphs. Specifically, created tag decay and tag attenuation functionality, as well as propagation policies to further mitigate false positive dependency explosions.

AWS News Search Engine
Built a basic news search engine using Java and AWS that downloads latest common crawl news articles, streams more than 100,000 documents to ElasticSearch index, and is capable of running on any machine provided there is a Java Virtual Machine implementation for it (Contact for code).

Ping and Traceroute
Coded an implementation of the popular ping and traceroute commands in Python, which can be used to test internet connection (from an ISP) and diagnose possible network ongestion between host and destination server.
